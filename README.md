### 关键特性

* 204 MHz Cortex-M4 + 2x Cortex-M0
* 264KB SRAM + 64KB ROM
* 一个6通道、12位高速ADC（ADCHS），具有DMA支持和80MSamples/s的数据转化率
* 以太网
* 两个HS USB
* LBGA256和TFBGA100封装
